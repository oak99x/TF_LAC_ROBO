Orientações sobre a forma de executar o programa:

1. Em caso de troca de automato, as definições no arquivo de entrada devem seguir o mesmo padrão;
    * ordem dos dados;
    * espaçamento;
    * uso dos simbolos especiais "(){},=";
    * o arquivo não deve ter linhas vazias, sejam elas no meio ou no final;

2. Para a insersão de novas palavras, o usuario deve abrir o arquivo "palavras.txt" e digitar as palavras uma abaixo do outra;
3. A palavra "VAZIO" pode ser representada como uma linha vazia ou com o símbolo " "(espaço);
4. Com arquivos preenchidos corretamente basta dar um Run no Main;
